# Social Media Tourism
MBA Research Project - Jain University

## Overview
This project investigates how social media data can be leveraged to predict tourism behavior and optimize targeted digital advertising strategies for the aviation industry. By analyzing customer digital footprints, travel-related interactions, and device preferences, predictive models were developed to identify high-propensity customers for travel services.

## Problem Statement
Create 2 different models for each type of device customers use to access social networking platforms to learn the customer behavior on travel-related pages, their recent travel check-ins, and their influence on others.  
- One model is for **Laptop users**.  
- Another model is for **Mobile users** (anything that is not a laptop is considered a mobile device).

## Objectives
- Build separate predictive models for laptop and mobile users.
- Analyze customer behavior on travel-related pages and check-ins.
- Optimize digital advertising costs by targeting only potential customers.
- Provide business and social opportunities through efficient marketing.

## Methodology
- Data collected from 11,760 unique customers via a social networking platform.
- Variables include likes, comments, reviews, check-ins, family/work status, and device type.
- Applied data wrangling, univariate and bivariate analysis, missing value treatment, and outlier handling.
- Built machine learning models with performance tuning for both laptop and mobile datasets.

## Models Used
- **Decision Tree**  
- **Random Forest**  
- **Gaussian Naive Bayes (GaussianNB)**  
- **Support Vector Machine (SVM)**  

## Key Findings
- Laptop users showed a higher purchase propensity (~25%) compared to mobile users (~15%).
- Following the company’s social media page increased purchase likelihood to ~40%.
- Preferred travel locations (beach, entertainment, hill stations) correlated with higher product adoption.
